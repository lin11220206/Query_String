# Query String

## 環境設置

需求：Ubuntu or Linux Mint

#### 安裝nodejs
`sudo apt-get update`
`sudo apt-get install nodejs`

#### 安裝npm
`sudo apt-get install npm`

## 設定伺服器

#### 安裝npm相依套件
`npm install`

#### 執行伺服器
`npm start`    (預設port為8080)
or
`PORT=[number] npm start` (自行設定port)

網頁將設於 http://localhost:port/
